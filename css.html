There are 3 ways to apply CSS styling to the HTML:

  1. 1. Styling inside the element tag itself:
    <h1 style="color: black; font-size: 15px;"></h1>
  2. 2. Styling inside the HTML file Using <style> tag:
    <style>
      h1 {
        color: black;
        font-size: 15px;
      }
    </style>
    You should place the <style> tag in the <head> tag.
  3. 3. Linking to external CSS file:
    <link rel="stylesheet" href="./location of file">

    This is the recommended method to style entire web-sites.

Selectors are used to target specific parts of the HTML code.

px/mm/cm – Fixed size, with no regard to the screen size. px recommended.

Relative:
vh & vw – View height & View width, units relative to screen size (viewport). used mainly for big containers like <body>.
% - percentage – Relative to father container.
em – For fonts. Relative to the font size of the closest ancestors.
rem – relative to the default font size of the root (of the main element - <HTML>)

  • Static – default.
  • Relative – moves relative to original position. stays in the flow of the page.
  • Absolute – relative to any ancestor who's not in a static position. gets out of the flow of the page.
  • Fixed – relative to viewport. meaning that it keeps its position even if the user scrolls up and down. Gets out of main flow.
  • Sticky – a combination of all the above. Stays in the flow. Remember to specify a location parameter (top/left ect...), otherwise he won't move if scrolled.

Deals with content that gets spilled out of its container.

Overflow: Visible; - default.
Overflow: Hidden; – wont show the parts that overflow.
Overflow: Scroll; – will force a scroll bar on all sides, regardless of the necessity of it.
Overflow: Auto; – will create a scroll only if the content get spilled in that direction.

Pseudo selectors allow you to control the style of elements under certain states.
for example when you hover over it, when you click it ect...
Pseudo elements we'v learned:

  • :hover

    = when the mouse is above the element.

  • :active

    = when you click the element.

  • :focus

    = for inputs. when you click on the input field.

  • :visited

    = for links, you can style a link differently when it was clicked once.

  • :checked

    for checkboxes / radios.

  • :nth-child()

    = selects a specific child of an element.

Transitions are used to animate the transition between two states of the same element.
For example if a transformation is applied when :hover, :active ect..
Syntax:

transition: transform 2s ease-in;

In this example we've set the transition to apply on the transform parameter,
The animation will be 2 seconds long,
And will have a Ease-in effect (The animation will start fast and end slowly).
Remember! you need to place the transition in the main CSS selector, NOT in the :hover Pseudo-class!

You can set the transition to apply on all the different parameters, using 'all':

transition: all 2s ease-in;


Or you can set on multiple parameters, like this:

transition: width 2s, height 4s;

This way you can set different timing for different parameters.
